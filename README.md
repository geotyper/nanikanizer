# nanikanizer
An neural network library for C++

## Features

### Expressions
  - Four arithmetic operations
  - Abs
  - Square
  - Square root
  - Sum
  - Norm
  - Squared norm
  - Sigmoid
  - Sigmoid cross entropy
  - ReLU
  - Matrix production
  - Convolution
  - Depth Concat

### Optimizers
  - SGD
  - AdaGrad
  - RMSProp
  - AdaDelta
  - Adam

## Requirements

### Supported Compilers
  - Clang (>= 3.7.0)
  - Visual Studio 2015

### External Libraries
  - Boost C++ Libraries (>= 1.60)
