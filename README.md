# nanikanizer
An neural network library for C++

## Features

### Expressions
  - Four Arithmetic Operations
  - Abs
  - Square
  - Square Root
  - Sum
  - Norm
  - Squared Norm
  - Sigmoid
  - Sigmoid Cross Entropy
  - ReLU
  - Matrix Production
  - Convolution
  - Depth Concat

### Optimizers
  - SGD
  - AdaGrad
  - RMSProp
  - AdaDelta
  - Adam

## Requirements

### Supported Compilers
  - Clang (>= 3.7.0)
  - Visual Studio 2015

### External Libraries
  - Boost C++ Libraries (>= 1.60)
